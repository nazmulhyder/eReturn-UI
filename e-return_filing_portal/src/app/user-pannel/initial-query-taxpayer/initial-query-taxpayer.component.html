Taxpayers who have paid any source tax, advance tax, or any amount of regular tax for 2021-2022 Assessment Year :

 1. Update your tax payment status using eReturn Ledger system. You will be diverted to the ledger system if you select YES to the questions given left side

 2. Verify the payment of tax (in relevant cases) using the ledger system. If any payment can not verified through the system, contact service center.

 3. Confirm the verified payment amount. The eReturn Ledger system will now divert you to eReturn system for return filing.
